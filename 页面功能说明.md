# EarthGlow 页面功能说明

## 页面概览

项目包含两个主要页面，都已集成完整的 AI 图片生成功能：

### 1. color1.html - 色彩交互界面
**主题：** "Time for an art makeover!"
**定位：** 以色彩为中心的艺术再创作工具

### 2. draw.html - 拼贴画创作界面
**主题：** "Weave Your Story"
**定位：** 拼贴、素描、场景构建工具

---

## 功能对比表

| 功能 | color1.html | draw.html |
|------|-------------|-----------|
| **AI 图片生成** | ✅ 完整支持 | ✅ 完整支持 |
| **颜色提取** | ✅ 5色自动提取 | ✅ 5色自动提取 |
| **历史记录** | ✅ 支持（带缩略图面板） | ⚠️ 支持（无UI面板）* |
| **下载功能** | ✅ 支持 | ✅ 支持 |
| **预设按钮** | ✅ 3个预设 | ✅ 3个预设 |
| **自定义提示词** | ✅ 支持 | ✅ 支持 |
| **灵感面板** | ✅ Inspiration + History | ❌ 无 |
| **素材面板** | ❌ 无 | ✅ Assets 面板 |
| **工具栏图标** | 4个（书签、取色、收藏、下载） | 9个（保存、调整、设置等） |

*注：draw.html 的历史记录功能可用（数据保存），但没有显示历史缩略图的 UI 面板。数据与 color1.html 共享。

## color1.html 详细功能

### 🎨 界面布局

```
┌─────────────┬───────────────────┬─────────────┐
│  左侧面板    │   中间画布区域     │  右侧聊天   │
├─────────────┼───────────────────┼─────────────┤
│ ○ 原始作品   │  ▣ 主画布         │  💬 AI对话  │
│ ● 5色点     │  ⭐ 收藏/下载     │  🔘 预设    │
│ 🎨 调色板   │  🖼️ 灵感面板      │  ⌨️ 输入框  │
│             │  📜 历史面板       │             │
└─────────────┴───────────────────┴─────────────┘
```

### ✨ 核心功能

1. **文本生成图片**
   - 使用 GRSAI nano-banana-fast 模型
   - 支持自定义提示词
   - 3个预设：Golden Hours / 80s Retro Disco / Morandi's Afternoon

2. **自动颜色提取**
   - K-means 聚类算法
   - 提取 5 种主导色
   - 显示在左侧色点区域
   - 鼠标悬停显示色码

3. **历史记录**
   - 最多保存 20 条
   - 显示在底部历史面板
   - 点击缩略图恢复
   - localStorage 持久化

4. **灵感面板**
   - 2张示例图片
   - 可用作参考（Phase 2 功能）

### 📍 访问地址
```
http://localhost:8000/color1.html
```

## draw.html 详细功能

### 🖌️ 界面布局

```
┌─────────────┬───────────────────┬─────────────┐
│  左侧面板    │   中间画布区域     │  右侧聊天   │
├─────────────┼───────────────────┼─────────────┤
│ ○ 原始作品   │  ▣ 主画布         │  💬 AI对话  │
│ ● 5色点     │  🎨 左侧工具栏    │  🔘 预设    │
│ 🎨 调色板   │  （6个图标）       │  ⌨️ 输入框  │
│             │  ⭐ 右侧操作栏    │             │
│             │  （3个图标）       │             │
│             │  🧩 素材面板       │             │
└─────────────┴───────────────────┴─────────────┘
```

### ✨ 核心功能

1. **文本生成图片**（与 color1.html 相同）
   - GRSAI API 集成
   - 自定义提示词
   - 相同的 3 个预设按钮

2. **自动颜色提取**（与 color1.html 相同）
   - K-means 算法
   - 5 色提取
   - 实时更新色点

3. **扩展工具栏**
   - **左侧工具（6个）：**
     - 💾 保存
     - 🔧 调整
     - ⚙️ 设置
     - 🖌️ 画笔
     - 🧽 橡皮擦
     - 🔷 形状

   - **右侧操作（3个）：**
     - 📋 复制
     - ⭐ 收藏
     - 📥 下载

4. **素材面板（Assets）**
   - 显示可用素材
   - 用于拼贴创作（未实现交互）

### 📍 访问地址
```
http://localhost:8000/draw.html
```

## 共享功能说明

### 💾 数据共享

两个页面使用**相同的 localStorage**，因此：
- ✅ 历史记录互通
- ✅ 配置设置共享
- ✅ 在 color1.html 生成的图片，draw.html 也能访问（通过 localStorage）

### 🔧 共用模块

两个页面共享以下 JavaScript 模块：
- `config.js` - API 配置
- `js/api-client.js` - API 客户端
- `js/color-extractor.js` - 颜色提取器
- `js/storage-manager.js` - 存储管理
- `js/ui-controller.js` - UI 控制器
- `js/utils.js` - 工具函数

### 🎨 相同的 API 配置

```javascript
{
  apiKey: 'your-api-key',
  baseURL: 'https://grsai.dakka.com.cn',
  model: 'nano-banana-fast',
  aspectRatio: '1:1',
  imageSize: '1K'
}
```

## 使用建议

### 何时使用 color1.html？

适合以下场景：
- 🎨 **色彩实验** - 想尝试不同配色方案
- 📜 **需要历史记录** - 需要查看和恢复之前的作品
- 🖼️ **需要灵感参考** - 有示例图片可以参考
- 🎯 **专注色彩** - 主要关注色彩变化和情绪表达

### 何时使用 draw.html？

适合以下场景：
- 🖌️ **创作工具** - 需要更多创作工具（未来实现）
- 🧩 **素材拼贴** - 计划使用素材面板（未来实现）
- 🎭 **场景构建** - "Weave Your Story" 主题更适合叙事
- 🔧 **扩展功能** - 需要更多工具栏选项（未来扩展）

## 功能限制

### color1.html 限制
- ❌ 没有素材面板
- ❌ 工具栏图标较少
- ⚠️ 灵感面板图片是静态的（无法作为参考上传）

### draw.html 限制
- ❌ 没有历史记录 UI 面板（功能可用但不可见）
- ❌ 没有灵感面板
- ⚠️ 工具栏图标仅占位（实际功能未实现）
- ⚠️ 素材面板仅展示（无交互功能）

## 调试技巧

### 在浏览器 Console 中查看历史记录

```javascript
// 查看所有历史（两个页面共享）
const history = JSON.parse(localStorage.getItem('earthglow_history') || '[]');
console.log('总历史记录:', history.length);
console.log('历史详情:', history);

// 查看当前页面
console.log('当前页面:', document.title);
console.log('App 实例:', window.app);
```

### 清空历史记录

```javascript
// 在任一页面的 Console 执行
window.app.storageManager.clearHistory();
location.reload();
```

### 手动触发生成

```javascript
// color1.html 或 draw.html
document.getElementById('promptInput').value = 'Beautiful sunset';
window.app.handlePromptSubmit();
```

## 技术架构

两个页面使用**完全相同的技术栈**：
- 📦 **前端框架:** 纯 Vanilla JavaScript
- 🎨 **样式:** 玻璃态设计（Glassmorphism）
- 🤖 **AI API:** GRSAI Nano Banana
- 🎨 **颜色算法:** K-means 聚类
- 💾 **存储:** localStorage (5MB)
- 📊 **状态管理:** UIController 类
